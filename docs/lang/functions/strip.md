# Function: `strip`

The `strip()` function creates a new string with any leading, and trailing whitespaces removed.

## Syntax:
```
<string>.strip()
```

## Example:
```javascript
var.original = "
  Test strip function!
"
var.stripped = var.original.strip()

case true:
    // Render as: 'Message: "Test strip function!"'
    return DEBUG, "Message: ""@var.stripped"""
```
